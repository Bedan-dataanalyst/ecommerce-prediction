\# 🛒 **Ecommerce Customer Intelligence \& Prediction System**



A complete end-to-end \*\***machine learning + data analytics + web app project**\*\* that analyzes ecommerce transactions, segments customers using RFM analysis, and predicts customer value using machine learning.



\---



\# 🚀 **Live Demo**

👉 https://ecommerce-prediction-2026.streamlit.app/



\---



\# 📊 **Project Overview**



This project transforms raw ecommerce transaction data into actionable business insights using:



\- Data Cleaning

\- Feature Engineering (RFM Analysis)

\- Customer Segmentation

\- Machine Learning Prediction Model

\- Interactive Streamlit Dashboard



\---



\# 🎯 Business Problem



Ecommerce businesses struggle to:

\- Identify high-value customers

\- Understand customer behavior

\- Reduce churn

\- Improve targeting and marketing efficiency



This system solves that by automatically classifying and analyzing customers.



\---



\# 🧠 Key Features



\## 1. Customer Segmentation (RFM Model)

\- Recency (how recent purchases were)

\- Frequency (how often customers buy)

\- Monetary (how much they spend)



Segments:

\- VIP Customers

\- Loyal Customers

\- New Customers

\- At Risk Customers



\---



\## 2. Machine Learning Model

\- Algorithm: Random Forest Classifier

\- Predicts: High Value vs Regular Customer

\- Input: Spending, Orders, Items

\- Output: Customer Value Classification



\---



\## 3. Interactive Dashboard

Built with Streamlit + Plotly:



\- KPI Metrics

\- Customer Segment Distribution

\- Revenue Analysis

\- Top Customers

\- RFM Behavior Map

\- Live Prediction Tool
